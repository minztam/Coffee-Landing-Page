## Giới thiệu

**Coffee Landing Page** là một trang web giới thiệu quán cà phê hiện đại, tinh tế, được thiết kế hoàn toàn bằng **HTML, CSS và một chút JavaScript**. Giao diện tập trung vào trải nghiệm người dùng với:

- Hiệu ứng **parallax scrolling** mượt mà  
- Hình ảnh chất lượng cao, tối ưu hiệu suất  
- Thiết kế **responsive** hoàn hảo trên mọi thiết bị  
- Tương tác nhẹ nhàng với **hover effects** và **smooth scroll**
